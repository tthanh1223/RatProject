Hướng dẫn chạy app:
- Đầu tiên cài .NET về máy (lên mạng tự cài)
- tạo môi trường ảo (venv) cho python dễ dùng
- pip install mấy cái trong app.py để chạy (t đang dùng python 3.14 còn mấy cái kia thì tao không chắc à)
